# Coding Reactor Physics

Numerical reactor physics solvers implemented in C++ and Python.

This repository contains lecture-style and research-oriented
implementations of the diamond difference method.

---

## 🔹 Features

- Two-group / multi-group neutron diffusion
- Finite Difference / Finite Element / DG methods
- Nonlinear Diffusion Acceleration (NDA)
- Krylov subspace solvers (BiCGSTAB, GMRES, etc.)
- Sparse matrix computation
- Python visualization and verification

---

## 🔹 Languages & Tools

- C++ (High-performance solvers)
- Python (Visualization & prototyping)
- Eigen / STL
- matplotlib / numpy
